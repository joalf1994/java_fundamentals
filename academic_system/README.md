# Java Academic System

## Objetivo
Sistema de gestión de alumnos desarrollado en Java para practicar programación orientada a objetos, colecciones en memoria y buenas prácticas de desarrollo.

## Tecnologías
- Java 21
- Git

## Conceptos implementados

- Programación Orientada a Objetos
- Collections (List, Set, ArrayList, HashSet)
- Encapsulamiento
- Herencia
- Polimorfismo

## Funcionalidades

- Registro de alumnos
- Registro de profesores
- Registro de cursos
- Registro de notas
- Registro de matriculas
- Consulta de alumnos
- Consulta de notas por alumno y curso
- Consulta de profesores
- Registro de cursos por profesor
